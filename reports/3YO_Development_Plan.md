## 📖 HOW TO READ THIS REPORT

**Conformation → Distance/Surface Profile:**
- **Humerus Length** = primary distance indicator (Short=sprinter, Medium=versatile, Long=router)
- **Lumbosacral** = rear drive power (Ideal/VG = strong acceleration)
- **Forehand** = front end quality (Ideal = premium movement)
- **Stifles** = knee height (High = quick turnover/sprint, Low = long stride/route)
- **Rear Triangle** = biomechanical balance (Balanced = versatile, Femur Shortest = power, Hip-to-Stifle = ground coverage)

**Race Shape:**
- **Start** = gate speed (Conservative/Normal/Aggressive)
- **Pace Instruction** = running style (Lead/Stalk/Midpack/Trail + Push/Restrain)
- **Effort** = intensity (Breezing = light, Handily = strong push)
